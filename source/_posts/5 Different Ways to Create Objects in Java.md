JAVA中，创建对象有5种方式。现在开发一遍利用依赖管理工具（例如Spring）进行创建。除此之外，还有其他的方式。

|         |            |
| ------------- | -----:|
| new                    | 构造函数调用 |
| class类的newInstance()  | 构造函数调用      |
| 构造函数的newInstance() | 构造函数调用      |
| clone()方法 | 没有构造函数      |
| 反序列化 | 没有函数调用      |
